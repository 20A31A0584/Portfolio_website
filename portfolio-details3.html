Project Description

Introduction:
The goal of this project is to develop a graphical user interface (GUI) based Rock, Paper, Scissors game using the Python programming language and the Tkinter module. The game will provide an interactive and engaging experience for users, allowing them to play against the computer in the classic hand game of Rock, Paper, Scissors.
Features and Functionality:
a. User Interface
: A visually appealing and intuitive user interface will be created using the Tkinter module. The interface will consist of buttons for the user to select their choice (rock, paper, or scissors) and display the results of each round.
b. Game Logic
: The game will incorporate the standard rules of Rock, Paper, Scissors, where rock beats scissors, scissors beats paper, and paper beats rock. The computer will generate its choice randomly, providing a fair challenge for the player.
c.Score Tracking
: The game will keep track of the user's score and display it on the interface. The score will be updated after each round, indicating the number of wins, losses, and ties.
